우승자가 2이상인데, 같은이름일때 무엇을 기준으로 판단?

# 미션: [자동차 경주]

## 기능 목록 정리

- 유저에게 입력받기

  - 자동차 이름 입력
    - 이름 길이는 5자 이하
    - 쉼표로 구분
  - 시도할 횟수

- 자동차들은 전진 또는 정지

  - 전진하는 조건
    - 0~9까지 랜덤값을 구함
    - 4이상일 경우 전진

- 최종 우승자 출력

  - 단독 작업자 안내 문구
    - ex) 최종 우승자 : pobi
  - 공동 작업자 안내 문구
    -ex) 최종 우승자 : pobi, jun

- 사용자가 잘못된 값을 입력한 경우 throw문 실행
  - [ERROR]
